### **1. Что такое Number Parsing (разбор чисел)?**

Это процесс преобразования текстового представления чисел (строк) в соответствующие числовые значения в Go.

---

### **2. Пакет для работы с преобразованием строк: `strconv`**

Для базовых типов используется пакет `strconv` (string conversion).

---

### **3. Пример 1: Преобразование строки в целое число с помощью `Atoi`**

**Код:**

```go
numStr := "12345"
num, err := strconv.Atoi(numStr)
if err != nil {
    fmt.Println("Error parsing value:", err)
}
fmt.Println(num)
```

**Что происходит:**

1. `Atoi` (Alphabet to Integer) преобразует строку в тип `int`.
2. Возвращает два значения:
   - `num` — число (тип `int`).
   - `err` — ошибка (если преобразование не удалось).
3. Проверяем ошибку: если `err != nil`, выводим сообщение об ошибке.
4. Выводим результат.

**Проверка, что это число:**

```go
fmt.Println(num + 1) // Выведет 12346
```

Если можно выполнить арифметическую операцию — это число.

---

### **4. Пример 2: `ParseInt` для чисел с указанием системы счисления и размера**

**Код:**

```go
numStr := "12345"
numInt, err := strconv.ParseInt(numStr, 10, 64)
if err != nil {
    fmt.Println("Error parsing value:", err)
}
fmt.Println("Parsed Integer:", numInt)
```

**Что происходит:**

1. `ParseInt` преобразует строку в `int64` с указанием:
   - `10` — система счисления (десятичная).
   - `64` — размер в битах (возвращает `int64`).
2. Возвращает число типа `int64` и ошибку.

---

### **5. Пример 3: `ParseFloat` для чисел с плавающей точкой**

**Код:**

```go
floatStr := "3.14"
floatVal, err := strconv.ParseFloat(floatStr, 64)
if err != nil {
    fmt.Println("Error parsing value:", err)
}
fmt.Printf("Parsed float: %.2f\n", floatVal)
```

**Что происходит:**

1. `ParseFloat` преобразует строку в `float64`.
2. Второй аргумент (`64`) — размер в битах.
3. Используем `Printf` для форматирования вывода (два знака после запятой).

---

### **6. Пример 4: Преобразование двоичной строки в десятичное число**

**Код:**

```go
binaryStr := "1010"
decimalVal, err := strconv.ParseInt(binaryStr, 2, 64)
if err != nil {
    fmt.Println("Error parsing binary value:", err)
    return
}
fmt.Println("Parsed binary to decimal:", decimalVal)
```

**Что происходит:**

1. `ParseInt` с системой счисления `2` (двоичная).
2. Двоичная строка `"1010"` преобразуется в десятичное число `10`.

**Как считается:**

- Позиции (справа налево, начиная с 0):
  - `0` \* 2^0 = 0
  - `1` \* 2^1 = 2
  - `0` \* 2^2 = 0
  - `1` \* 2^3 = 8
- Итог: 0 + 2 + 0 + 8 = 10.

---

### **7. Пример 5: Преобразование шестнадцатеричной строки**

**Код:**

```go
hexStr := "FF"
hexVal, err := strconv.ParseInt(hexStr, 16, 64)
if err != nil {
    fmt.Println("Error parsing hex value:", err)
    return
}
fmt.Println("Parsed hex to decimal:", hexVal) // Выведет 255
```

**Что происходит:**

1. Система счисления `16` (шестнадцатеричная).
2. `"FF"` в шестнадцатеричной системе = 255 в десятичной.

---

### **8. Пример 6: Обработка ошибок при некорректном вводе**

**Код:**

```go
invalidNum := "456ABC"
_, err := strconv.Atoi(invalidNum)
if err != nil {
    fmt.Println("Error parsing value:", err) // Ошибка: invalid syntax
}
```

**Что происходит:**

1. Строка содержит нечисловые символы (`ABC`).
2. `Atoi` возвращает ошибку, так как не может преобразовать строку в число.

---

### **Итог:**

- Для разбора чисел в Go используется пакет `strconv`.
- Основные функции:
  - `Atoi` — строка в `int`.
  - `ParseInt` — строка в целое число с указанием системы счисления и размера.
  - `ParseFloat` — строка в число с плавающей точкой.
- Всегда обрабатывайте ошибки, так как входные данные могут быть некорректными.
