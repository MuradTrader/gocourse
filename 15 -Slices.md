- `j=2`: `twoD[2][2] = 2 + 2 = 4`

- –†–µ–∑—É–ª—å—Ç–∞—Ç: `twoD[2] = [2, 3, 4]`

### –ò—Ç–æ–≥–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ `twoD`

–ü–æ—Å–ª–µ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è –≤—Å–µ–≥–æ –∫–æ–¥–∞, `twoD` –±—É–¥–µ—Ç —Å–æ–¥–µ—Ä–∂–∞—Ç—å:

### –í–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è

### –ö–ª—é—á–µ–≤—ã–µ –º–æ–º–µ–Ω—Ç—ã

1. **–î–∏–Ω–∞–º–∏—á–µ—Å–∫–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞**: –ö–∞–∂–¥—ã–π –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π —Å—Ä–µ–∑ –∏–º–µ–µ—Ç —Å–≤–æ—é –¥–ª–∏–Ω—É, –∫–æ—Ç–æ—Ä–∞—è –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –≤–æ –≤—Ä–µ–º—è –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è.

2. **–ò–Ω–∏—Ü–∏–∞–ª–∏–∑–∞—Ü–∏—è –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä**: –í–Ω–µ—à–Ω–∏–π —Å—Ä–µ–∑ –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä—É–µ—Ç—Å—è —Å –∑–∞–¥–∞–Ω–Ω–æ–π –¥–ª–∏–Ω–æ–π, –Ω–æ –∫–∞–∂–¥—ã–π –µ–≥–æ —ç–ª–µ–º–µ–Ω—Ç (–≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π —Å—Ä–µ–∑) –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä–æ–≤–∞–Ω –æ—Ç–¥–µ–ª—å–Ω–æ.

3. **–ù—É–ª–µ–≤—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è**: –ü—Ä–∏ —Å–æ–∑–¥–∞–Ω–∏–∏ —Å—Ä–µ–∑–∞ —Å –ø–æ–º–æ—â—å—é `make([]int, length)`, –≤—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∏–Ω–∏—Ü–∏–∞–ª–∏–∑–∏—Ä—É—é—Ç—Å—è –Ω—É–ª–µ–≤—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º —Ç–∏–ø–∞ (`0` –¥–ª—è `int`).

4. **–ò–Ω–¥–µ–∫—Å–∞—Ü–∏—è**: –û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –∏–Ω–¥–µ–∫—Å–∞—Ü–∏—è –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å `0`. –í–ª–æ–∂–µ–Ω–Ω—ã–µ —Ü–∏–∫–ª—ã –∏—Å–ø–æ–ª—å–∑—É—é—Ç –∏–Ω–¥–µ–∫—Å—ã `i` –∏ `j` –¥–ª—è –¥–æ—Å—Ç—É–ø–∞ –∫ —ç–ª–µ–º–µ–Ω—Ç–∞–º.

### –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–µ –æ–±—ä—è—Å–Ω–µ–Ω–∏–µ –æ—Ç –∞–≤—Ç–æ—Ä–∞ –∫—É—Ä—Å–∞

–ê–≤—Ç–æ—Ä –∫—É—Ä—Å–∞ –ø–æ–¥—á–µ—Ä–∫–∏–≤–∞–µ—Ç, —á—Ç–æ —Ç–∞–∫–æ–π —Å–ø–æ—Å–æ–± —Å–æ–∑–¥–∞–Ω–∏—è –¥–≤—É–º–µ—Ä–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä (—Å—Ä–µ–∑–æ–≤ —Å—Ä–µ–∑–æ–≤) –ø–æ–∑–≤–æ–ª—è–µ—Ç —Å–æ–∑–¥–∞–≤–∞—Ç—å "–∑—É–±—á–∞—Ç—ã–µ" —Å—Ç—Ä—É–∫—Ç—É—Ä—ã (jagged arrays), –≥–¥–µ –∫–∞–∂–¥—ã–π –≤–Ω—É—Ç—Ä–µ–Ω–Ω–∏–π —Å—Ä–µ–∑ –º–æ–∂–µ—Ç –∏–º–µ—Ç—å —Ä–∞–∑–Ω—É—é –¥–ª–∏–Ω—É. –≠—Ç–æ –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è –æ—Ç –º–Ω–æ–≥–æ–º–µ—Ä–Ω—ã—Ö –º–∞—Å—Å–∏–≤–æ–≤ (–∫–æ—Ç–æ—Ä—ã–µ –≤ Go –∏–º–µ—é—Ç —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω—ã–µ —Ä–∞–∑–º–µ—Ä—ã –Ω–∞ –≤—Å–µ—Ö —É—Ä–æ–≤–Ω—è—Ö).

–¢–∞–∫–∂–µ –∞–≤—Ç–æ—Ä –¥–æ–±–∞–≤–ª—è–µ—Ç, —á—Ç–æ –¥–ª—è –æ—Ç–ª–∞–¥–∫–∏ —Å–ª–æ–∂–Ω—ã—Ö –≤–ª–æ–∂–µ–Ω–Ω—ã—Ö —Ü–∏–∫–ª–æ–≤ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `fmt.Printf` –¥–ª—è –≤—ã–≤–æ–¥–∞ –∑–Ω–∞—á–µ–Ω–∏–π –∏–Ω–¥–µ–∫—Å–æ–≤ –∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ –∑–∞–ø–æ–ª–Ω–µ–Ω–∏—è, –∫–∞–∫ –æ–Ω —ç—Ç–æ —Å–¥–µ–ª–∞–ª –≤ —Å–≤–æ–µ–º –ø—Ä–∏–º–µ—Ä–µ —Å —Å–æ–æ–±—â–µ–Ω–∏—è–º–∏ "adding value ...".

–≠—Ç–æ—Ç –ø—Ä–∏–º–µ—Ä –¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É–µ—Ç –≥–∏–±–∫–æ—Å—Ç—å —Å—Ä–µ–∑–æ–≤ –≤ Go –∏ –∏—Ö –ø—Ä–∏–º–µ–Ω–∏–º–æ—Å—Ç—å –¥–ª—è –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è —Å–ª–æ–∂–Ω—ã—Ö —Å—Ç—Ä—É–∫—Ç—É—Ä –¥–∞–Ω–Ω—ã—Ö.

–ú—ã —É–∂–µ —Ä–∞–∑–±–∏—Ä–∞–ª–∏ —Å—Ä–µ–∑—ã, –Ω–æ —Ç–µ–ø–µ—Ä—å —É–≥–ª—É–±–∏–º—Å—è –≤ –∏—Ö –≤–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ –∏ —Å—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å –º–∞—Å—Å–∏–≤–∞–º–∏.

–í Go —Å—Ä–µ–∑—ã (slices) –∏ –º–∞—Å—Å–∏–≤—ã (arrays) ‚Äî —ç—Ç–æ –ø—Ä–∏–Ω—Ü–∏–ø–∏–∞–ª—å–Ω–æ —Ä–∞–∑–Ω—ã–µ —Ç–∏–ø—ã –¥–∞–Ω–Ω—ã—Ö.

### 1. –ú–∞—Å—Å–∏–≤—ã (Arrays)

- **–§–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–∞—è –¥–ª–∏–Ω–∞**: –î–ª–∏–Ω–∞ –º–∞—Å—Å–∏–≤–∞ ‚Äî —á–∞—Å—Ç—å –µ–≥–æ —Ç–∏–ø–∞. –ù–∞–ø—Ä–∏–º–µ—Ä, `[3]int` –∏ `[5]int` ‚Äî —Ä–∞–∑–Ω—ã–µ —Ç–∏–ø—ã.

- **–ó–Ω–∞—á–µ–Ω–∏–µ (value type)**: –ü—Ä–∏ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–∏ –º–∞—Å—Å–∏–≤–∞ –¥—Ä—É–≥–æ–π –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –∏–ª–∏ –ø–µ—Ä–µ–¥–∞—á–µ –≤ —Ñ—É–Ω–∫—Ü–∏—é —Å–æ–∑–¥–∞–µ—Ç—Å—è **–∫–æ–ø–∏—è** –≤—Å–µ–≥–æ –º–∞—Å—Å–∏–≤–∞.

- **–†–∞–∑–º–µ—Ä**: –û–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –Ω–∞ —ç—Ç–∞–ø–µ –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ –∏ –Ω–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–∑–º–µ–Ω–µ–Ω.

–ü—Ä–∏–º–µ—Ä:

```go

arr2 := arr1  // arr2 ‚Äî —ç—Ç–æ –ø–æ–ª–Ω–∞—è –∫–æ–ø–∏—è arr1. –ò–∑–º–µ–Ω–µ–Ω–∏—è –≤ arr2 –Ω–µ –∑–∞—Ç—Ä–æ–Ω—É—Ç arr1.

```

### 2. –°—Ä–µ–∑—ã (Slices)

- **–î–∏–Ω–∞–º–∏—á–µ—Å–∫–∞—è –¥–ª–∏–Ω–∞**: –°—Ä–µ–∑ –Ω–µ –∏–º–µ–µ—Ç —Ñ–∏–∫—Å–∏—Ä–æ–≤–∞–Ω–Ω–æ–π –¥–ª–∏–Ω—ã. –û–Ω —è–≤–ª—è–µ—Ç—Å—è "–æ–∫–Ω–æ–º" –≤ –±–∞–∑–æ–≤—ã–π –º–∞—Å—Å–∏–≤.

- **–°—Å—ã–ª–æ—á–Ω—ã–π —Ç–∏–ø (–ø–æ–¥ –∫–∞–ø–æ—Ç–æ–º)**: –°—Ä–µ–∑ ‚Äî —ç—Ç–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö, —Å–æ–¥–µ—Ä–∂–∞—â–∞—è:

- –£–∫–∞–∑–∞—Ç–µ–ª—å (pointer) –Ω–∞ –±–∞–∑–æ–≤—ã–π –º–∞—Å—Å–∏–≤.

- –î–ª–∏–Ω—É (length) ‚Äî —Ç–µ–∫—É—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ —Å—Ä–µ–∑–µ.

- –ï–º–∫–æ—Å—Ç—å (capacity) ‚Äî –æ–±—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –±–∞–∑–æ–≤–æ–º –º–∞—Å—Å–∏–≤–µ, –Ω–∞—á–∏–Ω–∞—è —Å –ø–µ—Ä–≤–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ —Å—Ä–µ–∑–∞.

–û–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ —Ç–∏–ø–∞ —Å—Ä–µ–∑–∞ (–ø—Ä–∏–º–µ—Ä–Ω–æ —Ç–∞–∫ –æ–Ω –≤—ã–≥–ª—è–¥–∏—Ç –≤ –∏—Å—Ö–æ–¥–Ω–∏–∫–∞—Ö Go, –Ω–æ –Ω–µ –¥–æ—Å—Ç—É–ø–Ω–æ –≤ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–æ–º –∫–æ–¥–µ):

```go

type slice struct {

array *[underlying_array_length]elementType

len   int

cap   int

}

```

### 3. –°–æ–∑–¥–∞–Ω–∏–µ —Å—Ä–µ–∑–∞

- **–ò–∑ –º–∞—Å—Å–∏–≤–∞**:

```go

```

–¢–µ–ø–µ—Ä—å `sl` —Å—Å—ã–ª–∞–µ—Ç—Å—è –Ω–∞ –±–∞–∑–æ–≤—ã–π –º–∞—Å—Å–∏–≤ `arr`, –Ω–∞—á–∏–Ω–∞—è —Å –∏–Ω–¥–µ–∫—Å–∞ 1.

- **–õ–∏—Ç–µ—Ä–∞–ª —Å—Ä–µ–∑–∞**:

```go

sl := []int{1,2,3}  // —Å–æ–∑–¥–∞–µ—Ç—Å—è –º–∞—Å—Å–∏–≤ [3]int{1,2,3} –∏ —Å—Ä–µ–∑, –∫–æ—Ç–æ—Ä—ã–π –Ω–∞ –Ω–µ–≥–æ —É–∫–∞–∑—ã–≤–∞–µ—Ç (len=3, cap=3)

```

- **–° –ø–æ–º–æ—â—å—é `make`**:

```go

```

### 4. –í–Ω—É—Ç—Ä–µ–Ω–Ω–µ–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ —Å—Ä–µ–∑–∞ (–ø–æ–¥ –∫–∞–ø–æ—Ç–æ–º)

- **–ë–∞–∑–æ–≤—ã–π –º–∞—Å—Å–∏–≤**: –í—Å–µ —Å—Ä–µ–∑—ã –ø–æ—Å—Ç—Ä–æ–µ–Ω—ã –ø–æ–≤–µ—Ä—Ö –º–∞—Å—Å–∏–≤–æ–≤. –ö–æ–≥–¥–∞ —Å–æ–∑–¥–∞–µ—Ç—Å—è —Å—Ä–µ–∑ (–∫—Ä–æ–º–µ nil-—Å—Ä–µ–∑–∞), –≤—ã–¥–µ–ª—è–µ—Ç—Å—è –ø–∞–º—è—Ç—å –ø–æ–¥ –º–∞—Å—Å–∏–≤.

- **–ó–∞–≥–æ–ª–æ–≤–æ–∫ —Å—Ä–µ–∑–∞ (slice header)**: –≠—Ç–æ —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ (—É–∫–∞–∑–∞—Ç–µ–ª—å, len, cap), –∫–æ—Ç–æ—Ä–∞—è –æ–ø–∏—Å—ã–≤–∞–µ—Ç —Å—Ä–µ–∑. –ü—Ä–∏ –ø–µ—Ä–µ–¥–∞—á–µ —Å—Ä–µ–∑–∞ –≤ —Ñ—É–Ω–∫—Ü–∏—é –∫–æ–ø–∏—Ä—É–µ—Ç—Å—è –∑–∞–≥–æ–ª–æ–≤–æ–∫, –Ω–æ –Ω–µ –±–∞–∑–æ–≤—ã–π –º–∞—Å—Å–∏–≤. –ü–æ—ç—Ç–æ–º—É –∏–∑–º–µ–Ω–µ–Ω–∏—è —ç–ª–µ–º–µ–Ω—Ç–æ–≤ —Å—Ä–µ–∑–∞ –≤ —Ñ—É–Ω–∫—Ü–∏–∏ –±—É–¥—É—Ç –≤–∏–¥–Ω—ã —Å–Ω–∞—Ä—É–∂–∏ (—Ç–∞–∫ –∫–∞–∫ —É–∫–∞–∑–∞—Ç–µ–ª—å –Ω–∞ –º–∞—Å—Å–∏–≤ –æ–¥–∏–Ω–∞–∫–æ–≤—ã–π). –ù–æ –µ—Å–ª–∏ –≤–Ω—É—Ç—Ä–∏ —Ñ—É–Ω–∫—Ü–∏–∏ –∏–∑–º–µ–Ω—è—é—Ç —Å–∞–º —Å—Ä–µ–∑ (–Ω–∞–ø—Ä–∏–º–µ—Ä, –¥–µ–ª–∞—é—Ç append –∏ –ø–µ—Ä–µ–∑–∞–ø–∏—Å—ã–≤–∞—é—Ç —Å—Ä–µ–∑), —Ç–æ —ç—Ç–æ –Ω–µ –≤–ª–∏—è–µ—Ç –Ω–∞ –≤–Ω–µ—à–Ω–∏–π —Å—Ä–µ–∑, –µ—Å–ª–∏ –Ω–µ –≤–µ—Ä–Ω—É—Ç—å –µ–≥–æ –∏ –Ω–µ –ø–µ—Ä–µ–ø—Ä–∏—Å–≤–æ–∏—Ç—å.

### 5. –û–ø–µ—Ä–∞—Ü–∏–∏ –Ω–∞–¥ —Å—Ä–µ–∑–∞–º–∏ –∏ –∏—Ö –ø–æ—Å–ª–µ–¥—Å—Ç–≤–∏—è

- **Append**:

```go

```

- –ï—Å–ª–∏ –≤ –±–∞–∑–æ–≤–æ–º –º–∞—Å—Å–∏–≤–µ –µ—Å—Ç—å –º–µ—Å—Ç–æ (cap > len), —Ç–æ —ç–ª–µ–º–µ–Ω—Ç –¥–æ–±–∞–≤–ª—è–µ—Ç—Å—è –≤ —Å–≤–æ–±–æ–¥–Ω–æ–µ –º–µ—Å—Ç–æ, len —É–≤–µ–ª–∏—á–∏–≤–∞–µ—Ç—Å—è –Ω–∞ 1.

- –ï—Å–ª–∏ –º–µ—Å—Ç–∞ –Ω–µ—Ç (cap == len), —Ç–æ:

1. –°–æ–∑–¥–∞–µ—Ç—Å—è –Ω–æ–≤—ã–π –±–∞–∑–æ–≤—ã–π –º–∞—Å—Å–∏–≤ (–æ–±—ã—á–Ω–æ —É–¥–≤–æ–µ–Ω–Ω–æ–π –µ–º–∫–æ—Å—Ç–∏, –Ω–æ –¥–ª—è –º–∞–ª–µ–Ω—å–∫–∏—Ö —Å—Ä–µ–∑–æ–≤ –∞–ª–≥–æ—Ä–∏—Ç–º —Ä–æ—Å—Ç–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å –∏–Ω—ã–º).

2. –í—Å–µ —ç–ª–µ–º–µ–Ω—Ç—ã –∫–æ–ø–∏—Ä—É—é—Ç—Å—è –≤ –Ω–æ–≤—ã–π –º–∞—Å—Å–∏–≤.

3. –î–æ–±–∞–≤–ª—è–µ—Ç—Å—è –Ω–æ–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç.

4. –í–æ–∑–≤—Ä–∞—â–∞–µ—Ç—Å—è —Å—Ä–µ–∑, –∫–æ—Ç–æ—Ä—ã–π —Å—Å—ã–ª–∞–µ—Ç—Å—è –Ω–∞ –Ω–æ–≤—ã–π –º–∞—Å—Å–∏–≤ (–∏ –∏–º–µ–µ—Ç –Ω–æ–≤—É—é len –∏ cap).

–í–∞–∂–Ω–æ: –ø–æ—Å–ª–µ –∫–æ–ø–∏—Ä–æ–≤–∞–Ω–∏—è –≤ –Ω–æ–≤—ã–π –º–∞—Å—Å–∏–≤ —Å—Ç–∞—Ä—ã–π –º–∞—Å—Å–∏–≤ –æ—Å—Ç–∞–µ—Ç—Å—è –≤ –ø–∞–º—è—Ç–∏, –Ω–æ –µ—Å–ª–∏ –Ω–∞ –Ω–µ–≥–æ –±–æ–ª—å—à–µ –Ω–µ—Ç —Å—Å—ã–ª–æ–∫, —Ç–æ –æ–Ω –±—É–¥–µ—Ç —É–¥–∞–ª–µ–Ω —Å–±–æ—Ä—â–∏–∫–æ–º –º—É—Å–æ—Ä–∞.

- **–°—Ä–µ–∑—ã (slicing)**:

```go

```

- `newSlice` –±—É–¥–µ—Ç –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ç–æ—Ç –∂–µ –±–∞–∑–æ–≤—ã–π –º–∞—Å—Å–∏–≤, —á—Ç–æ –∏ `sl`.
